## 二、策略框架：三重验证

```
新币出现
  ↓
[安全检查] → 貔貅/DEV持仓/流动性/freezeAuth → 不过直接跳过
  ↓ 过了
[链上信号] → 庄家钱包参与 + holder增速 + 买卖比 + 大单
  ↓
[情绪确认] → 推特提及 + 叙事热点
  ↓
评分≥60 → 买入$20
```

### 评分模型（100分制）
- **A. 安全基础 30分**（硬性，不过=淘汰）
  - 非貔貅(10) + DEV持仓≤5%(10) + 流动性≥$10k(5) + 非scam(5)
- **B. 链上信号 40分**（核心权重）
  - 庄家/聪明钱参与(15) + holder增长(10) + 买卖比≥3:1(10) + 大单(5)
- **C. 市场结构 15分**
  - bonding curve 20-70%(5) + 市值$50k-$5M(5) + 大盘不跌(5)
- **D. 情绪 15分**
  - 推特≥3条非bot提及(10) + 热门叙事(5)

## 四、庄家识别方法（关键能力）

### 要跟的庄家特征
- 低位横盘期分批买入（3天-2周）
- 分仓操作（大号→小号→多个中间号）
- 持有>3天才开始卖
- 阶梯慢卖（每次<20%）

### 要避的Scam特征
- 同一区块等额买入（bundler）
- 新钱包/burner（无交易历史）
- <1小时dump
- DEV立即消失

### 检测工具
- Scanner v5（已在跑，自建聪明钱库）
- getTokenLargestAccounts（持仓集中度）
- Helius parsedTransactions（交易历史）
- RugCheck/honeypot.is（安全审计）
- Union-Find分仓关联追踪（已实现）
